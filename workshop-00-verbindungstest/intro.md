# Verbindungstest

Willkommen beim ersten Test des Cloud-Workshop-Repositories.

Dieses kurze Szenario prüft, ob:

- das GitHub-Repository korrekt mit Killercoda verbunden ist,
- Markdown-Dateien geladen werden,
- Befehle im Terminal ausgeführt werden können,
- die automatische Aufgabenprüfung funktioniert.

Klicke auf **Start**, um den Test zu beginnen.
